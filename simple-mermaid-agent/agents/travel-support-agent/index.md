# Travel Support Agent

A customer support agent for SkyWays Travel Agency that handles:
- New travel bookings (flights, hotels, packages)
- Booking modifications (dates, passengers, upgrades)
- Cancellations and refunds
- General travel information and questions
- Customer complaints and issue resolution

## Purpose
Provide professional, empathetic customer support that resolves issues quickly while maintaining company policies and customer satisfaction.

## Key Features
- Multi-path workflow based on inquiry type
- Built-in escalation path for complex issues
- Comprehensive guardrails for each type of interaction
- Clear resolution authority and escalation criteria

## Agent Type
Graph-based agent using Mermaid workflow with per-node instructions.
